## 🧪 비공개 테스트 진행 방법

### 1단계: 트랙 생성 및 AAB 업로드
1. Google Play Console > **테스트 및 출시 > 비공개 테스트** 클릭
2. **"트랙 만들기"** 버튼
3. **`app-release.aab`** 파일 업로드
4. 출시 노트 입력 (예: "v1.0.1 첫 비공개 테스트")

### 2단계: 테스터 20명 등록
1. **"테스터" 탭 > 이메일 목록 만들기** 클릭
2. 목록 이름 지정 (예: "째깍 보물섬 테스터")
3. Gmail 주소 20개 입력 (한 줄에 하나씩)
4. 저장

### 3단계: 출시 시작 및 링크 공유
1. **"출시 검토 시작"** 버튼 클릭
2. 생성된 **참여 링크(opt-in URL)**를 복사
3. 카톡/문자로 테스터들에게 전송
4. 테스터들이 링크 → 앱 설치 → **14일간 삭제 안 하면 끝!**

### 테스터 모집 채널
| 방법 | 설명 |
|------|------|
| 가족/지인 | 부모님, 형제, 친구 등 안드로이드 유저의 Gmail |
| Reddit | r/betatesting, r/androidapps |
| 클리앙 | 개발 게시판 테스터 모집 글 |
| 디시인사이드 | 앱개발 갤러리 |

> 💡 **꿀팁**: 실질적으로 12~15명만 모아도 통과되는 경우가 많음
